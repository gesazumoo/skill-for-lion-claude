## 서비스 목적(최초에만 작성)

다양한 원데이 클래스를 탐색하고 신청할 수 있는 플랫폼을 만들어줘.

운동, 러닝, 수영, 스터디, 취미 같은 클래스를 사용자가 쉽게 찾아볼 수 있는 서비스야.

---

## 화면 목적

사용자가 서비스에 처음 들어왔을 때 추천 클래스와 마감 임박 클래스를 빠르게 둘러보고, 관심 있는 클래스를 찾을 수 있는 홈 화면을 구현해줘.

---

## 기존 화면과 연결(두번째 화면부터 작성)

최초 홈 화면 구현이므로 기존 화면과 연결은 없어.

---

## 화면 구성

### 1. Hero 영역

서비스의 첫인상을 보여주는 상단 영역을 만들어줘.

포함 요소:
- 메인 문구
- 서브 문구
- 검색창
- 카테고리 버튼

메인 문구:
"오늘 바로 참여 가능한 원데이 클래스"

카테고리:
- 전체
- 운동
- 러닝
- 수영
- 스터디
- 취미
- 클래스

---

### 2. 추천 클래스 영역

사용자에게 추천할 만한 클래스를 보여줘.

- 가로형 클래스 카드 사용
- 최소 6개의 클래스를 보여줘
- 클래스 리스트 형태로 구성

---

### 3. 마감 임박 클래스 영역

마감 날짜가 가까운 클래스를 따로 보여줘.

- 가로형 클래스 카드 사용
- 가로 스크롤 클래스 리스트 형태로 구성
- 마감 임박 기준은 glossary.md의 기준을 따라줘

---

### 4. 클래스 모집 등록 유도 영역

사용자가 직접 클래스를 등록하도록 유도하는 영역을 만들어줘.

포함 요소:
- 유도 문구
- 클래스 등록 버튼

문구:
"나만의 클래스를 직접 모집해보세요"

---

### 5. 하단 네비게이션

모바일 앱처럼 하단에 고정된 하단 네비게이션을 만들어줘.

메뉴:
- 홈
- 검색
- 등록
- 내정보

현재 화면은 홈 화면이므로 홈 메뉴를 active 상태로 표시해줘.

---

## 필요한 기능

이 화면에서 아래 기능들이 실제로 동작하게 구현해줘.

- 검색창 입력 기능
- 카테고리 필터 기능
- 추천 클래스 리스트 표시
- 마감 임박 클래스 가로 스크롤 표시
- 신청하기 버튼 클릭 기능
- 하단 네비게이션 표시
- 홈 메뉴 active 표시

신청하기 버튼을 클릭하면 우선 alert로 클래스 제목이 표시되게 해줘.
