# 검색 화면 구현

## 화면 목적

사용자가 클래스 검색을 하고 결과로 리스트를 볼 수 있는 화면.

---

## 기존 화면과 연결

- 홈 화면 검색창에서 키워드 입력 후 Enter → `/search?q=<query>` 이동
- 하단 네비게이션 검색 버튼 클릭 → `/search` 이동

---

## 화면 구성

- 상단 고정: 검색바 (pill 스타일) + 카테고리 필터 칩
- 하단: 세로형 클래스 카드 리스트 (세로 스크롤)
- 클래스 카드 클릭 시 → 클래스 상세보기 팝업 표시

---

## 구현된 컴포넌트

### ClassCardVertical.vue (신규)
세로형 클래스 카드 (glossary 기준):
- 썸네일 이미지 (상단)
- 카테고리, 제목, 날짜, 지역, 참가 인원, 가격, 신청하기 버튼
- 카드 전체 + 신청하기 버튼 클릭 시 `select` 이벤트 emit

### ClassDetail.vue (신규)
클래스 상세보기 팝업 (glossary 기준):
- Bottom sheet (Teleport to body)
- 썸네일, 제목, 카테고리, 가격, 날짜, 지역, 마감일, 참가 현황, 상세 설명
- 하단 신청하기 버튼 (현재 alert 처리)
