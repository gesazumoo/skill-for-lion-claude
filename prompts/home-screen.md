원데이 클래스를 탐색하고 신청할 수 있는 플랫폼 홈 화면을 만들어줘. 
# 홈 화면 구현

## 서비스 목적

다양한 원데이 클래스를 탐색하고 신청할 수 있는 플랫폼.
운동, 러닝, 수영, 스터디, 취미 같은 클래스를 사용자가 쉽게 찾아볼 수 있는 서비스.

---

## 화면 목적

사용자가 서비스에 처음 들어왔을 때 추천 클래스와 마감 임박 클래스를 빠르게 둘러보고, 관심 있는 클래스를 찾을 수 있는 홈 화면.

---

## 화면 구성

### 1. Hero 영역

서비스의 첫인상을 보여주는 상단 영역.

포함 요소:
- 메인 문구
- 서브 문구
- 검색창
- 카테고리 버튼

메인 문구:
"오늘 바로 참여 가능한 원데이 클래스"

카테고리:
- 전체
- 운동
- 러닝
- 수영
- 스터디
- 취미
- 클래스

---

### 2. 추천 클래스 영역

사용자에게 추천할 만한 클래스.

- 가로형 클래스 카드 사용
- 가로 스크롤 클래스 리스트 형태로 구성
- 최소 6개의 클래스

---

### 3. 마감 임박 클래스 영역

마감 날짜가 가까운 클래스를 따로 표시.

- 세로형 클래스 카드 사용
- 가로 스크롤 클래스 리스트 형태로 구성
- 마감 임박 기준은 glossary.md의 기준을 따름 (3일 이내)

---

### 4. 클래스 모집 등록 유도 영역

사용자가 직접 클래스를 등록하도록 유도하는 영역.

포함 요소:
- 유도 문구
- 클래스 등록 버튼

문구:
"나만의 클래스를 직접 모집해보세요"

---

### 5. 하단 네비게이션

모바일 앱처럼 하단에 고정.

메뉴:
- 홈
- 검색
- 등록
- 내정보

현재 화면은 홈 메뉴 active 상태.

---

## 필요한 기능

- 검색창 입력 기능
- 카테고리 필터 기능
- 추천 클래스 리스트 가로 스크롤 표시
- 마감 임박 클래스 리스트 가로 스크롤 표시
- 신청하기 버튼 클릭 기능 (alert로 클래스 제목 표시)
- 하단 네비게이션 표시 / 홈 메뉴 active 표시

---

## 구현 조건

- Nuxt3 사용
- TailwindCSS 사용
- Composition API 사용
- 모바일 반응형 우선
- 컴포넌트 분리해서 작성
- API 연결 X / DB 연결 X / 더미 데이터 사용 O
- 화면이 비어보이지 않도록 실제 서비스처럼 보이는 더미 데이터를 충분히 넣을 것
- 이후 화면에서 클래스 정보와 클래스 카드를 재사용할 수 있게 만들 것
