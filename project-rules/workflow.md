# 작업 완료 체크리스트

아래 작업을 했다면 반드시 체크리스트를 전부 이행해줘.
"필요시"나 "선택적"이 아니라, 해당 작업을 했으면 **무조건** 실행해줘.

---

## 체크리스트 적용 기준

아래 작업 중 하나라도 해당하면 체크리스트를 이행해줘:

- 새로운 화면 생성
- 새로운 기능 추가
- 데이터 구조 변경
- API / DB 연결
- 프로젝트 구조 변경

---

## 작업 완료 체크리스트

- [ ] CLAUDE.md 현재 프로젝트 상태 업데이트
- [ ] /prompts 폴더에 작업 프롬프트 저장

## /prompts 저장 규칙

사용자가 보낸 프롬프트를 **그대로** 저장해줘.
요약하거나 재작성하지 말 것. 사용자가 입력한 텍스트를 복사해서 저장하면 됨.

---

단순 디자인 수정이나 작은 UI 수정은 체크리스트 생략 가능.
